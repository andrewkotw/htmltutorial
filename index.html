🚀 瀏覽器是怎麼運作的？

💡 老師的小叮嚀：
瀏覽器（像 Chrome 或 Edge）就像一個「組裝工人」。它會讀取下面這段程式碼，將 HTML 和 CSS 結合在一起。你可以點擊下面按鈕，**複製整段程式碼**到「記事本」，存成「`myweb.html`」，然後用瀏覽器打開，你就能在自己的電腦上看到網頁了！
